Authentication

Register User

Method: POST

Endpoint: /api/register/

Description: Register a new user account.

Input:

{
    "username": "new_user",
    "email": "user@example.com",
    "password": "securepassword123"
}

Response (Success):

{
    "message": "User registered successfully!",
    "username": "new_user",
    "email": "user@example.com"
}

Login User

Method: POST

Endpoint: /api/login/

Description: Authenticate user and return access/refresh tokens.

Input:

{
    "email": "user@example.com",
    "password": "securepassword123"
}

Response (Success):

{
    "access": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refresh": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Logout User

Method: POST

Endpoint: /api/logout/

Description: Invalidate the user's refresh token.

Headers:

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "message": "Logout successful!"
}

Refresh Access Token

Method: POST

Endpoint: /api/token/refresh/

Description: Get new access token using refresh token.

Input:

{
    "refresh": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Response (Success):

{
    "access": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}

Posts Management

Create Post

Method: POST

Endpoint: /api/posts/create/

Description: Create a new post.

Headers:

{
    "Authorization": "Bearer <access_token>"
}

Input (FormData):

{
    "content": "This is my new post!",
    "image": <file>  (optional)
}

Response (Success):

{
    "id": 1,
    "author": "new_user",
    "content": "This is my new post!",
    "image": "http://example.com/media/posts/image.jpg",
    "created_at": "2025-01-17T15:00:00Z",
    "likes_count": 0,
    "comments_count": 0
}

List Posts

Method: GET

Endpoint: /api/posts/

Description: Get all posts (paginated).

Headers: (optional)

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "count": 2,
    "next": null,
    "previous": null,
    "results": [
        {
            "id": 1,
            "author": "user1",
            "content": "First post content",
            "image": null,
            "created_at": "2025-01-16T12:00:00Z",
            "likes_count": 3,
            "comments_count": 2,
            "is_liked": false
        },
        {
            "id": 2,
            "author": "user2",

            "created_at": "2025-01-15T10:30:00Z",

Delete Post

Method: DELETE

Endpoint: /api/posts/delete/<int:pk>/

Description: Delete a post (only by author or admin).

Headers:

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "message": "Post deleted successfully!"
}

Comments Management

Create Comment

Method: POST

Endpoint: /api/posts/<int:post_id>/comments/create/

Description: Add a comment to a post.

Headers:

{
    "Authorization": "Bearer <access_token>",
    "Content-Type": "application/json"
}

Input:

{
    "content": "This is a great post!"
}

Response (Success):

{
    "id": 1,
    "post": 1,
    "author": "commenter_user",
    "content": "This is a great post!",
    "created_at": "2025-01-17T15:30:00Z"
}

Delete Comment

Method: DELETE

Endpoint: /api/comments/delete/<int:pk>/

Description: Delete a comment (only by author or admin).

Headers:

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "message": "Comment deleted successfully!"
}

Likes Management

Like a Post

Method: POST

Endpoint: /api/posts/<int:post_id>/like/

Description: Like a post.

Headers:

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "message": "Post liked successfully!"
}

Unlike a Post

Method: DELETE

Endpoint: /api/likes/delete/<int:pk>/

Description: Remove a like from a post.

Headers:

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "message": "Like removed successfully!"
}

User Profile

Get User Profile

Method: GET

Endpoint: /api/profile/ (current user) or /api/profile/<int:user_id>/

Description: Get user profile information.

Headers: (optional for public profiles)

{
    "Authorization": "Bearer <access_token>"
}

Response (Success):

{
    "id": 1,
    "username": "test_user",
    "email": "user@example.com",
    "bio": "Hello world!",
    "profile_picture": "http://example.com/media/profiles/user.jpg",
    "posts_count": 5,
    "followers_count": 10,
    "following_count": 3,
    "is_following": false
}
